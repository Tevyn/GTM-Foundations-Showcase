# Competitive Brief: Verkada in Global Financial Services Firm Deal

## Deal Context

We're in a mid-stage evaluation with a **Fortune 500 financial services company** (~12,000 employees, 40+ branch and office locations across North America). The VP of Corporate Security is leading the evaluation. They currently run **Genetec Security Center** as their VMS with **Software House** access control across all sites. Verkada entered the deal after the prospect attended a Verkada-sponsored security conference panel last quarter. The prospect is evaluating both platforms for AI-powered video analytics and threat detection capabilities. Verkada has already completed an on-site demo at one branch location.

---

## Where We Win Against Verkada

- **Camera-agnostic deployment:** Ambient.ai works with their existing Hikvision and Bosch cameras across all 40+ sites. Verkada requires proprietary cameras, meaning the prospect would need a full hardware swap to get Verkada's AI capabilities.
- **Access Intelligence (video-PACS correlation):** Ambient.ai correlates video with their Software House access control data to reduce false alarms by **95%**. Verkada does not offer native video-access control correlation.
- **150+ verified threat signatures:** Ambient.ai's Threat Detection library covers a significantly wider range of security scenarios than Verkada's basic person/motion detection.
- **Advanced Forensics with AI-powered search:** Ambient.ai allows search by person appearance, object, or activity type across all cameras. Verkada's search capabilities are more limited.
- **Edge-optimized VLM (Pulsar):** Purpose-built Vision-Language Model for physical security -- Verkada has no equivalent.
- **Enterprise compliance:** Ambient.ai is **GDPR and SOC 2 Type II certified**. Verkada has had well-publicized security breaches in the past.

---

## Where Verkada Wins

- **Simpler pitch for greenfield:** If a prospect has no existing camera infrastructure, Verkada's all-in-one hardware + software bundle is an easier story to tell.
- **Single-vendor simplicity:** Verkada provides cameras, cloud dashboard, and analytics in one package -- less integration work on the surface.
- **Brand awareness in mid-market:** Verkada has strong marketing presence and name recognition, especially with facilities and IT buyers who may not come from a physical security background.

---

## The Value Wedge

Ambient.ai deploys as a cloud-delivered AI overlay on existing infrastructure. Verkada requires proprietary hardware. For a financial services organization with 40+ locations already running Genetec and Software House, the cost and disruption of a Verkada hardware swap would be significant. Ambient.ai delivers AI intelligence capabilities -- threat detection, false alarm reduction, AI-powered forensics -- without replacing a single camera or access control panel.

Key features to emphasize:
- No rip-and-replace vs. full hardware swap
- 95% false alarm reduction via Access Intelligence
- 150+ threat signatures vs. basic motion/person detection
- AI-powered forensic search across existing cameras
- GDPR + SOC 2 Type II compliance
